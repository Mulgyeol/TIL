# CSS정의

  • Cascading Style Sheet
    • cascading : 연속해서 떨어지는 느낌
      • 브라우저에는 기본적으로 정의된 스타일이 있다.
      • 직접 지정한 스타일이 없다면 브라우저의 스타일로 넘어간다는 의미이다.

# 웹사이트 스타일링 분류

  • Author Style : 작성한.css 파일
  • User Style : 브라우저상에서 사용자가 (선택해서) 지정한 스타일
  • Browser : 브라우저에서 기본적으로 정의 된 스타일
  • 우선순위 : Author Style => User Style => Browser
    • 이렇게 적용되는 것이 Cascading
    • 이 Cascading의 연결고리를 끊어내는 것 : ! important (Bad Smell)
      • 안쓰는게 좋다.

# 선택자(Selectors)

  • Universal : *
  • type : Tag ex)div
  • Id : #id ex)#name
  • Class : .class ex).member_name
  • State : :
  • Attribute : []

# CSS 적용방법

  property: value;
}

# 적용 우선순위

  • Tag에 가깝게 설정할 수록 우선순위가 높다.
    • 속성,클래스 => 태그 => 전체(*)
    • div태그는 block level element이다. 따라서, div태그에 내용이 없을 경우, 영역이 지정되지 않아서 해당 태그에 적용한 css가 적용이 안된다.
      • 넓이와 높이를 따로 지정해주면 배경색 설정등은 적용된다.

# Block Level Element 살펴보기

  • padding : 컨텐츠 안에 들어가는 spacing
    • padding-top, padding-right 등 각각 사이드에 따로 사이즈를 넣을 수 있다.
    • padding: 20px 15px 10px 5px
      • 위쪽부터 시계방향으로 적용된다.
      • 위 : 20px, 오른쪽 : 15px, 아래 : 10px, 왼쪽 : 5px
    • padding: 20px 0px
      • 위아래 / 양 옆
      • 위 : 20px, 오른쪽 : 0px, 아래 : 20px, 왼쪽 : 0px
  • margin : 컨텐츠 밖에 들어가는 spacing
    • margin도 padding과 같은 방식으로 적용 가능
  • border : 경계값
    • border: 2px solid red;
      • 차례로 width, style, color
      • 아래 세 줄을 한 줄로 지정한 것이다.
    • border-width: 2px;
    • border-style: solid;
    • border-color: pink;
div {
  width: 100px;
  height: 100px;
  padding: 20px;
  margin: 20px;
  /* border: 2px solid red; 아래 세 줄 과 같은 의미 */
  border-width: 2px;
  border-style: solid;
  border-color: pink;
  background: yellow;
}
